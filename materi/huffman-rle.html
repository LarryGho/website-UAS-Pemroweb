Huffman, RLE, dan Kuantisasi

Pendahuluan

Huffman, Run Length Encoding (RLE), dan kuantisasi merupakan teknik penting dalam proses pemampatan citra digital. Teknik ini digunakan untuk mengurangi redundansi data citra.

Tujuan Pembelajaran

  • Memahami algoritma Huffman
  • Mengetahui konsep Run Length Encoding
  • Memahami peran kuantisasi dalam kompresi

Algoritma Huffman

Huffman adalah metode kompresi lossless yang menggunakan kode variabel berdasarkan frekuensi kemunculan data.

Run Length Encoding (RLE)

RLE menyimpan data berdasarkan jumlah kemunculan berurutan dari nilai piksel yang sama.

Kuantisasi

Kuantisasi mengurangi jumlah tingkat intensitas citra untuk mengurangi ukuran data.

Ringkasan

Huffman dan RLE merupakan metode kompresi lossless, sedangkan kuantisasi digunakan pada kompresi lossy.
